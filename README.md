# 极客版流媒体视频合成引擎

一套抛弃了传统 MP4 拼接，专为 Web 网页设计的 HLS (.m3u8) 视频切片生成工具。

## 核心优势
- **拒绝合并卡死**：采用 HLS 标准，自动生成碎片化 `.ts` 流文件和 `playlist.m3u8`，合并阶段零耗时。
- **全动态调整**：Web 面板实时所见即所得，支持文字、注音、翻译、词性标签、水印、序列号的动态排版与颜色/透明度配置。
- **网页秒切片加载**：生成的文件完美契合现代网站播放架构，自动支持按需加载与平滑缓冲。

## 快速运行

1. 安装依赖：`pip install flask pillow edge-tts`
2. 将背景图存放在 `static/bg.png`
3. 将 MiSans 字体放入 `ttf/`
4. 运行 `python app.py` 并在浏览器打开 `http://127.0.0.1:5000`

## 如何部署到你的网站
直接将 `static/output/` 文件夹下生成的所有内容 (`playlist.m3u8` 和所有 `.ts` 视频) 拖进你的 Web 服务器/CDN 中。
使用前端常用的 `hls.js` 或是直接在 `<video>` 标签读取 `playlist.m3u8` 即可极速播放！
